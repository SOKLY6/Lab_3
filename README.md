# Lab 3 - CLI для работы с алгоритмами и структурами данных

Интерактивное CLI-приложение на Python с реализацией различных алгоритмов сортировки, математических функций и структур данных (стек, очередь).

## Описание

Проект представляет собой интерактивный интерфейс командной строки для:
- Вычисления чисел Фибоначчи (итеративно и рекурсивно)
- Вычисления факториала (итеративно и рекурсивно)
- Сортировки массивов различными алгоритмами
- Работы со стеком и очередью
- Бенчмаркинга производительности алгоритмов

## Возможности

### Математические функции
- **Числа Фибоначчи**: итеративный и рекурсивный методы
- **Факториал**: итеративный и рекурсивный методы

### Алгоритмы сортировки
- Пузырьковая сортировка (Bubble Sort)
- Быстрая сортировка (Quick Sort)
- Пирамидальная сортировка (Heap Sort)
- Сортировка подсчётом (Counting Sort)
- Блочная сортировка (Bucket Sort)
- Поразрядная сортировка (Radix Sort)

### Структуры данных
- **Стек**: операции push, pop, peek, min, length, is-empty
- **Очередь**: операции enqueue, dequeue, front, length, is-empty

### Генерация тестовых данных
- Случайные целые числа
- Случайные вещественные числа
- Обратно отсортированные массивы
- Массивы с дубликатами
- Почти отсортированные массивы
- Пользовательские массивы

### Бенчмаркинг
Автоматическое измерение времени выполнения всех операций с выводом в миллисекундах.
